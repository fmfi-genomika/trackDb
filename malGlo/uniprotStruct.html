Description

This track shows structural information about regions of specific proteins that were parsed out from UniProt database (http://www.uniprot.org/). These regions were mapped to corresponding regions of the genome. MalGlo uniprot data can be found at http://www.uniprot.org/proteomes/UP000008837.

Structure types

This track shows only few specific structure types:
  • coiled-coil region
  • beta
  • helix
  • turn
Each region is annotated with its specific type in the name that is shown in the browser, for instance, position NW_001849855.1:3129-3242 displays coiled-coil region. These structure types were based on "feature type" in UniProt.

Setup

The full setup of the track can be found at https://github.com/fmfi-genomika/genomikaMalGlo/wiki/uniprotStruct-and-uniprotAnnot-tracks.
