Description

This track shows protein annotations for specific regions that were parsed out from uniprot database (http://www.uniprot.org/). These regions were mapped to corresponding regions of the genome. MalGlo uniprot data can be found at http://www.uniprot.org/proteomes/UP000008837.

Annotations types

Each region shows its type with additional information about protein (if were available). Current track contain these types:
  • active site
  • binding site
  • calcium-binding region
  • chain
  • compositionally biased region
  • disulfide bond
  • DNA-binding region
  • domain
  • glycosylation site
  • initiator methionine
  • intramembrane region
  • lipid moiety-binding region
  • metal ion-binding site
  • modified residue
  • non-terminal residue
  • nucleotide phosphate-binding region
  • region of interest
  • repeat
  • short sequence motif
  • signal peptide
  • site
  • topological domain
  • transmembrane region
  • zinc finger region
Each track is annotated with its region type and additional information (format [name] | [information]), for instance, position NW_001849855.1:6316-6504 displays domain | T-SNARE coiled-coil homology. These annotations were based on "feature type" in UniProt.

Setup

The full setup of the track can be found at https://github.com/fmfi-genomika/genomikaMalGlo/wiki/uniprotStruct-and-uniprotAnnot-tracks.
